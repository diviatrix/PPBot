Welcome to PPBot

To start using 1337.plus system you need to use Telegram

Start with Telegram

Available commands:

  • /go - registers user with defaultUser data and Telegram ID
  • /roll - Gives user random PP of the Day or show recieved if timeout is not reached
  • /top - use with any number to check its stats, or if it is added
  • /me - shows user data stats All bot answers are replies to request message
  • /pp - use with any number to check its stats, or if it is added
  • /add - sends a suggestion to admin
  • /rar - shows rarities and counts of pp in database for each
  • /bonus - recieve bonus if it is added
  • /a - a?
  • /deleteme - delete your account
